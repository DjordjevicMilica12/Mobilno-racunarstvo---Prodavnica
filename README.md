# Aplikacija za kupovinu odeće

## Kratak opis aplikacije
Aplikacija predstavlja prototip prodavnice gde je omogućeno da korisnik pregleda celokupnu ponudu i potom kreira porudžbinu ukoliko to želi.

Aplikacija je razvijena uz pomoć Angular i Ionic framework-a, dok se za backend koristi Firebase uz pomoć koje se čuvaju proizvodi, registrovani korisnici kao i njihove porudžbine.

### Omogućene funkcionalnosti:
- Registracija i prijava korisnika
- Kreiranje porudžbine
- Izmena podataka prilikom porudžbine
- Otkazivanje porudžbine

## Uputstvo za instalaciju
1. **Kloniranje repozitorijuma**

   Potrebno je da klonirate repozitorijum na lokalni računar:

   ```bash
   git clone https://github.com/DjordjevicMilica12/Mobilno-racunarstvo---Prodavnica.git
2. **Ulazak u projekat**

    Potom je potrebno da uđete u projekat koristeći komandu::

   ```bash
   cd Prodavnica

3. **Instalacija paketa**

    Da bi aplikacija mogla da se pokrene, neophodno je da se instaliraju svi paketi, to se radi sledećom komandom:
  
   ```bash
   npm install
    ```

4. **Pokretanje aplikacije**

    Kada je sve instalirano, aplikacija se može pokrenuti sa:

   ```bash
   ionic serve
   ```

### Korišćene tehnologije:
- Angular - frontend framework
- Ionic - framework za razvoj hibridnih mobilnih aplikacija
- Firebase - backend platforma za čuvanje podataka
